📱 Gen-Z Social Media & Mental Health — Data Analysis

A data science investigation into social media usage patterns and mental health
outcomes among Generation Z (ages 18–26).

## 🔍 Key Finding
Strong negative correlation between daily usage hours and mental health score  
**(r = -0.774, p < 0.001)**

## 📊 Techniques Used
- Exploratory Data Analysis & Visualization
- Pearson Correlation
- Independent Samples t-Test
- One-Way ANOVA

## 🛠 Stack
`Python` · `pandas` · `numpy` · `matplotlib` · `seaborn` · `scipy`
## 📁 Dataset
Synthetic dataset (AI-generated) simulating 500 Gen-Z social media users  
across 5 countries with 9 variables.
